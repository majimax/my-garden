# 🌱 我的花园 - 游戏说明书

> **版本：** 2.1  
> **更新：** 2026-04-23  
> **游戏时长：** 15-25分钟

---

## 一句话介绍

一家人种花园，每人有自己的任务（种花/种菜/种果），有人想让它开花，有人偷偷使坏。

---

## 🎯 目标

让**属于你任务的植物**成功开花，得分最高者获胜。

---

## 👤 任务卡（随机分配，每人不同）

| 人物 | 任务 | 每张得分 |
|------|------|----------|
| 👨 爸爸 | 水果 + 蔬菜 | 2分 |
| 👩 妈妈 | 花卉 + 水果 | 2分 |
| 👴 爷爷 | 花卉 + 蔬菜 | 2分 |
| 👵 奶奶 | 蔬菜 | 3分 |
| 👦 哥哥 | 水果 | 3分 |
| 👧 姐姐 | 花卉 | 3分 |

---

## 🔄 每回合流程

### 1️⃣ 抽牌
从主牌库抽1张加入手牌

### 2️⃣ 执行主操作（3选1，只能选一个）

| 操作 | 怎么做 |
|------|--------|
| 🌱 **种幼苗** | 选手牌里的幼苗卡 → 点击有种子、无幼苗的区域 |
| 🍃 **盖叶子** | 选手牌里的叶子卡 → 点击已有幼苗的区域 |
| ✅ **开花** | 点击高亮区域（需种子+幼苗+叶子齐全）→ 翻开种子看结果 |

### 3️⃣ 道具（可选，用1次/回合）

| 道具 | 效果 |
|------|------|
| ✂️ 剪刀 | 移除场上任意幼苗/叶子 |
| 🔧 铲子 | 查看种子，可选择换成另一张 |
| 💧 水桶 | 从弃牌区回收1张幼苗/叶子 |

### 4️⃣ 弃牌
手牌超过5张需丢掉多余卡

### 5️⃣ 结束回合

---

## ✅ 开花规则

种子卡写明了需要什么组合：

```
种子要求 = 幼苗类型（单子叶/双子叶） + 叶子形状（卵形/披针/心形/掌状）
```

- 实际放的卡匹配要求 → ✅ **成功**
- 不匹配 → ❌ **失败**

## 🏆 计分

所有区域开花后，统计成功区域：

| 任务类型 | 得分 |
|----------|------|
| 爸爸（水果+蔬菜） | 2分/张 |
| 妈妈（花卉+水果） | 2分/张 |
| 爷爷（花卉+蔬菜） | 2分/张 |
| 奶奶（蔬菜） | 3分/张 |
| 哥哥（水果） | 3分/张 |
| 姐姐（花卉） | 3分/张 |

**示例：** 场上成功：花卉2个、水果1个、蔬菜1个

- 奶奶（蔬菜3分）：1 × 3 = **3分**
- 姐姐（花卉3分）：2 × 3 = **6分**

## 💡 胜利技巧

1. **记住已出的卡** — 幼苗/叶子打出后会进弃牌区，有助于推断还剩什么
2. **用剪刀干扰对手** — 对手快要开花时，剪掉关键卡
3. **用铲子换走好种子** — 发现对手种子是它需要的，就换走
4. **控制手牌数量** — 保持4-5张，避免被迫弃掉关键卡
