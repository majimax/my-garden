## 🌾 缘起

爷爷蹲在地头，眯着眼打量了一圈，特别满意：

*"这丝瓜秧长得不赖，都是我的功劳。"*

我刚想说话，爸爸从旁边走过来，皱着眉看了一会儿：

*"不对啊，这地里上周我埋的是西瓜籽啊。"*

爷爷一愣，不说话了。

我正想说点什么，奶奶从屋里出来了，看见我们仨站在那儿，神秘兮兮把我拉到一边，小声说：

*"前天我翻地种了点南瓜——杂草太多了，顺手给你清了。"*

我张了张嘴，看看爷爷，看看爸爸，再看看奶奶。

**一块地，三个人，三种瓜。**

谁也没告诉谁。

最后长出来的——丝瓜、西瓜、南瓜，傻傻分不清。

稀里糊涂种了一季，谁都没商量，还都觉得自己种的是对的。

**这就是我家的菜地。**

**也是我做这款游戏的原因。**

## 一句话介绍

一家人种花园，每人有自己的任务（种花/种菜/种果），有人想让它开花，有人偷偷使坏。

---

## 🎯 目标

让**属于你任务的植物**成功开花，得分最高者获胜。

## 🏁 游戏准备

1. **分牌库**：将种子卡洗混背面朝上放在一摞；幼苗卡、叶子卡和道具卡洗混背面朝上放在另一摞作为公共牌库
2. **抽取人物卡**：将人物卡洗混，每个玩家抽取一张，不要展示，其他没被选的人物卡背面朝下收好
3. **确认任务**：每个玩家确认自己人物卡上的任务
4. **埋种子**：每个玩家从种子牌库抽取4张种子卡，选择其中2张背面朝上放置在桌面中央，作为自己埋下的种子，其他种子卡洗混放在种子牌库下面
5. **发手牌**：每个玩家从公共牌库抽取5张手牌
6. **决定起始玩家**：由最近拍摄过花的玩家先开始回合

---

## 🔄 每回合流程

### 1️⃣ 抽牌
从公共牌库抽1张加入手牌

### 2️⃣ 执行主操作（3选1，只能选一个）

| 操作 | 怎么做 |
|------|--------|
| 🌱 **种幼苗** | 选手牌里的幼苗卡 → 点击有种子、无幼苗的区域 |
| 🍃 **盖叶子** | 选手牌里的叶子卡 → 点击已有幼苗的区域 |
| ✅ **开花** | 点击高亮区域（需种子+幼苗+叶子齐全）→ 翻开种子看结果 |

### 3️⃣ 道具（可选，用1次/回合）

| 道具 | 效果 |
|------|------|
| ✂️ 剪刀 | 移除场上任意幼苗/叶子 |
| 🔧 铲子 | 查看种子，可选择换成另一张 |
| 💧 水桶 | 从弃牌区回收1张幼苗/叶子 |

### 4️⃣ 弃牌
手牌超过5张需丢掉多余卡

### 5️⃣ 结束回合

## ✅ 开花规则

种子卡写明了需要什么组合：

```
种子要求 = 幼苗类型（单子叶/双子叶） + 叶子形状（卵形/披针/心形/掌状）
```

- 实际放的卡匹配要求 → ✅ **成功**
- 不匹配 → ❌ **失败**

|----------|------|

## 💡 胜利技巧

1. **记住已出的卡** — 幼苗/叶子打出后会进弃牌区，有助于推断还剩什么
2. **用剪刀干扰对手** — 对手快要开花时，剪掉关键卡
3. **用铲子换走好种子** — 发现对手种子是它需要的，就换走
4. **控制手牌数量** — 保持4-5张，避免被迫弃掉关键卡
